# Lua 示例五：算法与数据结构

## 示例1：0/1背包状态转移（伪代码）
```text
for each item:
    for w from W down to weight:
        dp[w] = max(dp[w], dp[w-weight] + value)
```
讲解：逆序遍历容量避免同物品重复使用。
